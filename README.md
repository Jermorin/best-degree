#best-degree

##firt use

- `npm install`
- launch your mongo and elasticSearch db
- edit `environment.json` with your own settings
- import seed data with `npm run import`

##use

- `npm run start`
- GET `/api/health-check`

##stack

- node with es6 syntax
- mongodb
- elasticSearch
